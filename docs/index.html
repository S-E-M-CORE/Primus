Primus - Der glühende Colt

Primus ist ein Mitgliederverwaltungssystem für einen Schützenverein. Es ermöglicht das Verwalten von Mitgliedern.

About

Was ist Primus?

Primus ist ein modernes Mitgliederverwaltungssystem, das im Rahmen eines Schulprojekts entwickelt wurde. Es soll die veraltete Excel-97-Version für den fiktiven Schützenverein "Der glühende Colt" ersetzen.

REST-Api

Das Projekt ist in Frontend und Backend unterteilt. Die Kommunikation findet mittels einer REST-API statt

C++ Backend

Das Backend des Projekts ist vollständig in C++ geschrieben. Hierfür wurde das Framework Oatpp verwendet

Responsiv

Die mitgelieferte GUI ist mittels Bootstrap vollständig responsiv und ermöglicht somit auch eine Nutzung mittels Smartphone

Features

Die Features dieses Mitgliederverwaltungs-Tools

CRUD-Operationen für Mitglieder
Kompakte Übersicht
Anwesenheitslisten von Mitgliedern führen
Einteilen in Abteilungen
Automatisches Errechnen des Beitrags von Mitgliedern
Erfassen, ob ein Mitglied eine Waffe kaufen darf
